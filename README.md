# Seenons API
## Description

Simple API to demonstrate knowledge of TypeScript and NestJS

## Installation

```bash
$ npm install
```

## Running the app
Create a .env file in the root of the project and fill the following fields.
```
MONGO_URL=mongodb://mongodb_uri
PORT=3000 or whatever port you prefer
```
```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
